## Tools

### `test_client_start`
Запускает процесс тест‑клиента 1С и подключается к нему.
Строка запуска формируется автоматически: `КаталогПрограммы()` + `СтрокаСоединенияИнформационнойБазы()` + ключи клиента.

Параметры:
- `port` — `TPort` для подключения (опционально, по умолчанию `1538`).

Результат:
- `started` — булево.
- `connected` — булево.
- `details` — строка или структура.

### `test_client_stop`
Разрывает соединение и пытается закрыть тест‑клиент.

Параметры:
- отсутствуют.

Результат:
- `stopped` — булево.

### `find`
Ищет окна, формы и элементы интерфейса тест‑клиента.

Параметры:
- `root_uri` — URI корневого объекта поиска (опционально, по умолчанию поиск от `ТестируемоеПриложение`).
- `type` — `window|form|group|button|table|field|decoration` (опционально).
- `title` — маска заголовка с поддержкой `*` и `?` (опционально).
- `name` — маска имени объекта с поддержкой `*` и `?` (опционально).
- `timeout_sec` — таймаут поиска в секундах (опционально).

Результат:
- массив объектов `type`, `title`, `name`, `uri`.

Ошибки:
- `-32602` — некорректные параметры или неподдерживаемый URI.
- `UI_NOT_READY` — тест‑клиент не подключен.
- `UI_NOT_FOUND` — объект не найден.
- `UI_AMBIGUOUS_OBJECT` — URI разрешается неоднозначно.
- `-32603` — ошибка вызова методов тест‑клиента.

### `activate`
Активизирует окно, форму или control по URI.

Параметры:
- `uri` — `window://...`, `form://...` или `control://...`.

Результат:
- объект `type`, `title`, `name`, `uri`.

Ошибки:
- `-32602`, `UI_NOT_READY`, `UI_NOT_FOUND`, `UI_AMBIGUOUS_OBJECT`, `-32603`.

### `close`
Закрывает окно или форму по URI.

Параметры:
- `uri` — `window://...` или `form://...`.

Результат:
- объект `type`, `title`, `name`, `uri` закрытого окна или формы.

Ошибки:
- `ACTION_DENIED` — попытка закрыть `control://...`.
- `-32602`, `UI_NOT_READY`, `UI_NOT_FOUND`, `UI_AMBIGUOUS_OBJECT`, `-32603`.

### `open_form`
Открывает форму через `ExecuteCommand` активного окна.

Параметры:
- `target` — строка команды; на текущем этапе это навигационная ссылка.

Результат:
- `target` — исходная строка команды.
- `window_uri` — URI активного окна после выполнения команды.
- `form_uri` — URI открытой формы, если новая форма определена однозначно.

Ошибки:
- `-32602`, `UI_NOT_READY`, `-32603`.

## Resources
- `window://active` — активное окно.
- `window://{path}` — конкретное окно по навигационной ссылке.
- `form://{name}` — форма по имени формы.
- `control://{form}/{name}` — элемент формы по имени формы и имени элемента.
- Шаблоны ресурсов публикуются через `resources/templates/list`.
- Поддерживаемый subset RFC 6570 ограничен выражениями `{var}`.

## Уведомления
- `notifications/ui/changed` — изменения UI.
- `notifications/tools/listChanged`.
- `notifications/resources/listChanged`.
- `notifications/prompts/listChanged`.

## Заглушки
- Форматы `formId`/`elementId`.
- Расширенные действия (drag‑drop, выбор из списка, таблицы).
